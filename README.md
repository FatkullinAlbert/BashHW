# BashHW
Домашнее задание по теме: "Скрипты Bash"
Задание:
Написать скрипт для CRON, который раз в час формирует отчёт и отправляет его на заданную почту.
Отчёт должен содержать:
IP-адреса с наибольшим числом запросов (с момента последнего запуска);
Запрашиваемые URL с наибольшим числом запросов (с момента последнего запуска);
Ошибки веб-сервера/приложения (с момента последнего запуска);
HTTP-коды ответов с указанием их количества (с момента последнего запуска).
Скрипт должен предотвращать одновременный запуск нескольких копий, до его завершения.
Для начала нам нужно понять что устанавливать:
Для проверяемого ПО, которую я буду смотреть я решил выбрать nginx:
apt install nginx
Так же нам нужно ПО для работы с почтой:
apt install mailutils|
И планирововщик задач CRON:
sudo apt install cron
С этого момента мы можем написать скрипт.
В самом скрипте, адресата я выбрал локальную машину, в которой работал.
Мы можем несколько раз походить на нашу ВМ curl запросом
curl http://localhost
Чтобы появились логи и было что отправлять
В скрипте есть строка (практически на самом вверху)
EMAIL="$USER"
После указания адресата и написания самого скрипта мы можем сделать его исполняемым и выполнить.
Если на нашем mail появилось сообщение-отчёт о нашем nginx отчёте, то скрипт работает.
Теперь надо поработать с cron:
Надо открыть от root наш планировщик задач:
sudo crontab -e
И добавить в конец скрипта строку:
0 * * * * /usr/local/bin/web_report.sh
Теперь он будет отправлять отчёт в 00 минут каждого часа
Задание выполнено!
